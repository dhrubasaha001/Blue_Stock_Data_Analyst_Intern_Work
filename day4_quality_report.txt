DAY 4 QUALITY REPORT

OBJECTIVE

Perform advanced mutual fund performance analytics using historical NAV data and benchmark indices to evaluate fund performance through financial risk and return metrics.

## DATASETS USED

* nav_history_clean.csv
* scheme_performance_clean.csv
* benchmark_indices.csv
* fund_master.csv

---

## PERFORMANCE METRICS COMPUTED

1. Daily Returns

* Calculated daily percentage returns for all mutual fund schemes.
* Verified return distribution using histogram visualization.

2. Compound Annual Growth Rate (CAGR)

* Computed CAGR using beginning and ending NAV values.
* Generated comparison table across all schemes.

3. Sharpe Ratio

* Calculated annualized Sharpe Ratio using:
  Risk-Free Rate = 6.5%
* Ranked all funds based on risk-adjusted performance.

4. Sortino Ratio

* Calculated downside-risk-adjusted returns.
* Considered only negative daily return observations.

5. Alpha and Beta

* Performed linear regression against the NIFTY100 benchmark.
* Estimated annualized Alpha and market Beta for each fund.

6. Maximum Drawdown

* Computed running maximum NAV.
* Measured worst historical drawdown for every fund.

7. Composite Fund Scorecard

* Combined multiple performance indicators into a weighted score:
  • 30% Three-Year Return
  • 25% Sharpe Ratio
  • 20% Alpha
  • 15% Expense Ratio (Inverse)
  • 10% Maximum Drawdown (Inverse)

8. Benchmark Comparison

* Compared top-performing funds with NIFTY50 and NIFTY100.
* Generated benchmark comparison visualization.

## KEY FINDINGS

1. Daily return distributions were centered near zero with moderate volatility.

2. CAGR values highlighted long-term wealth creation across multiple funds.

3. Funds with higher Sharpe Ratios delivered stronger risk-adjusted performance.

4. Sortino Ratios demonstrated downside risk protection for several schemes.

5. Positive Alpha values indicated benchmark outperformance.

6. Beta values reflected varying sensitivity to market movements.

7. Maximum Drawdown effectively captured downside market risk.

8. Composite scoring provided balanced fund rankings beyond simple return metrics.
